🔑 Anthropic API Key

DART uses Claude (by Anthropic) to perform deep NLP-based review analysis. Your API key is stored only in your browser's local storage — it is never sent to any server other than Anthropic's own API endpoint.

Get a free API key at: console.anthropic.com →

ℹ️ How the AI works

When you analyze a product URL or paste review text, DART:

  • Fetches the product page via a free CORS proxy (allorigins.win)
  • Uses Claude AI to extract product name, image, and review text
  • Runs NLP analysis detecting sentiment manipulation, AI-generated patterns, keyword stuffing, and linguistic uniformity
  • Returns a 0–100 deception score with LOW / MEDIUM / HIGH risk classification
  • Provides per-review breakdown and actionable consumer recommendation

🔒 Privacy

Your API key is stored only in localStorage in your browser. Review text is sent directly from your browser to api.anthropic.com — no other server sees it. All analysis logs are stored locally in your browser only and can be cleared at any time.
