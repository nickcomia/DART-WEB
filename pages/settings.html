🔐 API Configuration

To enable AI analysis, please enter your API key. The key is stored locally in your browser and never shared with any third party.

⚡ How DART works

DART uses a private deep learning engine to analyze review text and images. Your API key enables secure access to the AI models. No review data is stored on our servers – everything remains in your browser.

  • Text analysis: NLP for fake patterns and sentiment
  • Image analysis: Authenticity check for visual content
  • Behavioral scoring: Rating vs. text consistency
